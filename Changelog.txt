Changelog of TXLWizard
======================

v1.7 (2016-06-07)
-----------------
* Add function `TXLWizard.TXLWriter.TXLWriter.ImportTXL()` for importing existing TXL files and e.g. add labels.
* Add `TXLWizard.ShapeLibrary.LabelArray` for labelling an array of objects.
* Add `Precision` attribute in `TXLWizard.TXLWriter.TXLWriter` for specifying the TXL float precision
* Add Presentation Slides (PDF)
* Improved font rendering in `TXLWizard.ShapeLibrary.Label` for improved E-Beam Performance

v1.6 (2016-05-20)
-----------------
* First Public Release
* Add Documentation
